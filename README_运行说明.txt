我的教室 VR A-Frame 修正版

运行方法：
1. 解压整个文件夹。
2. 用 VS Code 打开文件夹。
3. 右键 index.html，选择 Open with Live Server。
4. 浏览器地址一般是：http://127.0.0.1:5500/index.html

操作方法：
- 鼠标拖动：转动视角
- W/A/S/D：前后左右移动
- 右下角 VR：进入 VR 模式
- 对准地面蓝色圆点约 2 秒：视角移动到对应位置

这版修正了打开后只看到白墙的问题：
之前相机初始位置在后墙外面，所以视线被墙挡住；现在初始位置放在教室内部后排，打开后能直接看到教室桌椅和投影墙。
